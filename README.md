# 嗯 毕设


### 概述
* 这篇毕设主要是worktile[http://www.worktile.com](http://www.worktile.com "worktile")仿站，只截取了最最简单最最基本的功能，有时间的话再来一步一步完善吧
* 语言为express + mongoose + jQuery的组合，express跟mongoose 都不是很熟悉，学了一点点，按照自己的想法胡乱的用
